# Simple Calculator

This is a basic calculator project implemented in C. It performs fundamental arithmetic operations and is designed to demonstrate basic command-line interface (CLI) interactions and C programming fundamentals.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Interactive menu loop

## Usage
1. Compile the program:
   `gcc -Wall -Werror -Wextra -pedantic calculator.c -o calculator`
2. Run the program:
   `./calculator`
3. Follow the on-screen menu to choose an operation.
